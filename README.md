## Connections for Wemos Lolin32 lite 

| GPIO | TFT   | Description         |
| ---: | :---- | :------------------ |
| 23   | SDA   | MOSI Hardware SPI   |
| 18   | SCK   | CLK  Hardware SPI   |
| 15   | CS    | CS                  |
|  2   | DC    | DC                  |
|  4   | RES   | Reset               |
| 21   | BLK   | 3.3V ( or PWM-Pin ) |
|      | VCC   | 3.3V                |
|      | GND   | GND                 |
